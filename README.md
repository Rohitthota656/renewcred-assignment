# RenewCred CMS Assignment

## Overview

This project is a full-stack Content Management System (CMS) built as part of the RenewCred Frontend Engineering Assignment.

## Tech Stack

### Frontend
- Next.js
- Redux Toolkit
- React

### Backend
- Express.js
- MongoDB Atlas
- JWT Authentication

## Features

- Admin Login
- JWT Authentication
- Admin Dashboard
- Add Standards
- View Standards
- Dynamic Content Management
- MongoDB Integration

## Project Structure

```
FrontendAssignment/
├── frontend/
├── backend/
```

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend folder.

Example:

```
MONGODB_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
PORT=5000
```
